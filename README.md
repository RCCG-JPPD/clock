# Draggable Electron Window

This Electron application creates a draggable window displaying the content of a specified URL.

## Prerequisites

Make sure you have Node.js installed. You can download it from [nodejs.org](https://nodejs.org/).

## Installation

1. Clone the repository or download the source code.
2. Open a terminal and navigate to the directory containing the source code.

```bash
cd path/to/your/project
```

3. Install Electron globally if you haven't already:

```bash
npm install -g electron
```

## Running the Application

1. Navigate to the project directory (if not already there).
2. Run the Electron application:

```bash
electron autoloader.js
```


### How to Use

1. **Download the code**: Clone the repository or download the zip file of the source code.
2. **Navigate to the directory**: Open a terminal and navigate to the directory containing the source code.
3. **Install Electron**: Run `npm install -g electron` to install Electron globally if you haven't already.
4. **Run the application**: Execute `electron main.js` to start the application.
